Quick explanation of the code and app:
-	It loads an Airbnb dataset containing listings from European cities
-	Users can filter accommodation by their preferences on the:
o	City
o	Capacity (number of guests)
o	Maximum price
o	Type of accommodation
-	Finally, the app:
o	Display a data table of matching listings
o	Shows their location on an interactive map
o	Plots a bar chart comparing the average prices by city and accommodation type

The objective is to help users to easily identify what place corresponds the most to what their needs.
